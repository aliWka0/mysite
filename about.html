Ben Kimim?

Yazılım, oyun geliştirme, gömülü sistemler ve network alanlarında projeler üreten bir geliştiriciyim.

Unity ve Unreal Engine 5 ile mobil/PC oyunlar geliştiriyor, Arduino ve PLC sistemleriyle otomasyon ve robotik projeler tasarlıyorum.

Aynı zamanda Flask + Keras tabanlı yapay zeka entegrasyonlu web uygulamaları geliştirerek uçtan uca proje üretme deneyimine sahibim.

Kullandığım Teknolojiler

Oyun Geliştirme

  • Unity (C#): 3D mobil oyunlar, fizik tabanlı mekanikler, korku oyunları (Backrooms)
  • Unreal Engine 5 (C++): Multiplayer FPS sistemleri, network mantığı, kod tabanlı proje mimarisi

Yazılım Geliştirme

  • Flask + Keras: Yapay zeka entegrasyonlu web uygulamaları, kripto fiyat tahmin sistemi
  • Visual Studio: Form tabanlı masaüstü uygulamalar (login, rol sistemi, CRUD, arama)
  • HTML/CSS/JS: Web tabanlı oyun ve simülasyonlar, GitHub Pages ile yayınlama

Gömülü Sistem & Otomasyon

  • Arduino: 8 servo spider robot, mesafe sensörü entegrasyonu, pin planlama
  • Siemens S7-1200 (TIA Portal v17): PLC tabanlı otopark otomasyon sistemi

Network & Altyapı

  • Cisco Packet Tracer: OSPF routing, ACL, NAT, Telnet kısıtlama, CDP/LLDP
  • GitHub Pages: Domain bağlama, SEO sorun tespiti
